# Okul Kulüp Üyelik Sistemi

 Java ile geliştirilmiş bir okul kulüp üyelik yönetim uygulaması.

## 📌 Proje Açıklaması
Bu proje, okul kulüplerine ait öğrenci bilgilerini nesne tabanlı programlama kullanarak yönetir. Üyeler bir dizi içerisinde tutulur ve dosyaya kaydedilir.

## Özellikler
- Kulüp üyelerini saklama (dizi ile)
- Üyeleri listeleme
- Kulüplere göre gruplandırarak görüntüleme
- Verileri dosyaya kaydetme (FileWriter)

##  Kullanılan Teknolojiler
- Java  
- Nesne Tabanlı Programlama (OOP)  
- Dosya İşlemleri (I/O)

## 🎯 Amaç
Bu proje, Java’da sınıf yapısı ve dosya işlemleri konularını öğrenmek için hazırlanmıştır.
